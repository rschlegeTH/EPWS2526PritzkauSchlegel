# PoC Simulations-Erfolg
## Es gilt zu beweisen, dass das System erfolgreich Prokrastination, durch Elemente A vs. B auslösen kann.
**Exit Kriterium:**

Über 70% der Nutzer geben in Interviews eine Zustimmung von über 3 auf einer Skala von 1-5, bei der Aussage: "Ich habe mich durch Element B abgelenkt gefühlt."

**Fail Kriterium:**
Über 30% der Nutzer geben in Interviews einen Score von 3 oder weniger, bei der Aussage: "Ich habe mich durch Element B abgelenkt gefühlt."

Fallback: PoC: Prokrastination durch Konsequenzen simulieren (Multiple Spielenden)
