# Installationsanleitung Stand: 15.01.2026

## Windows 
Im Windows Ordner die EP-Game.exe herunterladen und runnen.
P kann verwendet werden, um die Zeit zu pausieren.

## Mac
Hier wird leider die Development Build verwendet werden müssen.

## Development Build (in engine)
Um die aktuelle Variante zu nutzen, muss das Projekt aus dem Development Ordner und die [Godot Engine 4.5](https://godotengine.org/download/archive/4.5.1-stable/) Standard-Version heruntergeladen werden. Die Engine ist verpackt ca 100 mB groß.
Im Anschluss muss die Godot Engin(ohne C++) geöffnet werden und der Import-Knopf oben links getätigt werden. Dann kann der Ordner und die darin enthaltene 'project.godot'-Datei ausgewählt werden.

Im Projekt reicht oben rechts auf das Play-Zeichen zu drücken und der Prototyp kann gestaret werden.
